Who can use a CRA user ID and password

To register with this option, you must have:

  • A social insurance number (SIN)
  • Filed your taxes in the last 2 years

You can use a CRA user ID and password to sign in to:

  • My Account
  • My Business Account
  • Represent a Client

Manage your CRA user ID and password

Create, change or recover your CRA user ID and password, and what to do if your user ID and password has been revoked.

Create a strong CRA user ID and password

When you create your CRA user ID and password, we recommend that you:

  • Make them easy for you to remember and hard for others to guess
  • Avoid using personal information such as your name, SIN, mailing or email address
  • Always use unique passwords rather than reusing the same password for different accounts or systems
  • Keep this information secure and do not share it with anyone

The only special characters you can use in your user ID and password are:

  • Dot (.)
  • Dash (-)
  • Underscore (_)
  • Apostrophe (‘)

Creating your CRA User ID

Your CRA user ID must contain between 8 and 16 characters with no spaces, and can contain up to seven numbers.

Each CRA user ID must be unique. You will receive a message if the CRA user ID you create has already been used.

Creating your password

Your password must contain between 8 and 64 characters, including:

  • At least one upper-case letter
  • At least one lower-case letter
  • At least one number
  • No space
  • No accented characters

You cannot use more than 4 consecutive, identical characters.

Change your CRA user ID or password

You can change your user ID or password after signing in to your account.

My Account
  1. Sign in to your CRA My Account
  2. Select Profile
  3. Scroll to Security options
  4. Select Continue
My Business Account
  1. Sign in to your My Business Account
  2. Select Profile
  3. Scroll to CRA security options, Online preferences
  4. Select Continue
Represent a Client
  1. Sign in to Represent a Client
  2. Select Review and update
  3. Select RepID information
  4. Select Manage your CRA security options
Lost or forgotten CRA user ID or password

Forgotten user ID

You will need all of the following:

  • Your social insurance number (SIN)
  • Your date of birth
  • The amount you entered on line 15000 of your income tax and benefit return from the current or previous tax year

To recover your CRA user ID:

  1. Go to CRA accounts
  2. Select your account
  3. Select CRA user ID and password from the sign-in options
  4. Select Forgot your user ID?

The next time you sign in, you will need to answer 4 of your 5 CRA security questions.

Forgotten password

If you have forgotten your CRA password:

  1. Go to CRA accounts
  2. Select your account
  3. Select CRA user ID and password from the sign-in options
  4. Select Forgot your password?

You can create a new password if you correctly answer the security questions you chose when you registered.

For more information: Create a strong CRA user ID and password

Revoked CRA user ID and password

You may have received a notification that your user ID and password have been revoked. If that happens, you can regain access to your CRA account in the following ways:

Option 1
Use a different sign in method:

If you have one, you may be able to sign in by using a different CRA user ID and password, Interac® sign-in service, or a provincial digital ID to access your account.

Option 2
Register with a new CRA user ID:

You may need to contact us to register with a new CRA user ID and password.

Once registered, you will have to re-validate your identity with Interac document verification service or a CRA security code.

How to regain access to your Careers candidate profile

If your CRA user ID and password for the Candidate Profile service have been revoked, you may complete a candidate profile recovery. This will allow the new CRA user ID and password you create during the recovery process to be linked to your existing candidate profile.

Learn more about protecting your personal information: Keep your CRA account secure

Choose your security questions

You will be asked to choose 5 security questions that we will use to validate your identity in different situations, for example, when signing in, if you forget your user ID, or if you sign in from a new device.

Remember your answers

Your answers should be easy for you to remember but hard for others to guess. Answers that can change over time or questions that have more than one answer can be hard to remember.

Sign in without having to answer a security question

Once you have signed in to your account, you will have the option of selecting Skip this step on this computer. When you make this selection, a persistent cookie will be added to your browser and you will be remembered on this computer. After this, each time you sign in, you will not have to answer one of your security questions.

You should only "Skip this step on this computer" for your personal computer and not one that is shared by others, for example, at public libraries.

Change your sign-in option

You must always sign in to your account with the option you registered with.

If you registered with a CRA user ID and password and want to change your sign-in option, you can sign in to your account and revoke your credential so you can register with an Interac sign in partner instead.

You can also contact us by phone to revoke your credential so you can register again.

Locked account

If you enter the wrong CRA user ID or password too many times, your account will become locked. If this happens, you will have to contact us by phone to have your account unlocked.
